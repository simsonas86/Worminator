# Worminator

Worminator is a Twitch chat raffle bot for coaching raffles.


## Testing

See [tests/README.md](tests/README.md) for test suites, commands, and developer testing guidance.

## Database Schema

### users

| Column | Type | Description |
|--------|------|-------------|
| user_id | bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY | Primary key |
| twitch_id | bigint | id of twitch account |
| discord_id | bigint | id of discord account |
| username | varchar(50) | Discord username |
| authority_level | int | member (0), mod (1), admin (2), superadmin (3), system (-1) |
| created_at | timestamptz | When the user was first seen |

### coaching_sessions

| Column | Type | Description |
|--------|------|-------------|
| session_id | bigint GENERATED ALWAYS AS IDENTITY | Primary key |
| user_id | bigint | foreign key (users) |
| session_datetime | timestamptz | When the session occurs |
| created_at | timestamptz | When the record was created |

### raffles

| Column | Type | Description |
|--------|------|-------------|
| raffle_id | bigint GENERATED ALWAYS AS IDENTITY | Primary key |
| status | varchar(20) | `open`, `drawn`, `resolved`, `cancelled`, `no winner` |
| duration_seconds | int | how long the raffle is open for in seconds |
| starts_at | timestamptz | Raffle start time |
| ends_at | timestamptz | Raffle end time |
| winner_id | bigint | foreign key (users) of winner, nullable |
| created_at | timestamptz | When the raffle was created |

### ticket_events

| Column | Type | Description |
| --- | --- | --- |
| event_id | bigint GENERATED ALWAYS AS IDENTITY | Primary key |
| user_id | bigint | foreign key (users) |
| event_type | varchar(20) | `issued`, `redeem`, `claim`, `raffle_win`, `raffle_loss` |
| amount | int | delta tickets gained/lost |
| raffle_id | bigint | foreign key (raffles), nullable |
| issued_by | bigint | foreign key (users) |
| created_at | timestamptz | When the ticket event happened |
