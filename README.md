## Getting Started<a id="getting-started"></a>

### Requirements

- Python 3.10+
- PostgresSQL database
- Git
- uv

### Installation

Clone the repository:

```bash
git clone https://github.com/simsonas86/Worminator.git
cd Worminator
```

Create virtual environment and install dependancies:

```bash
uv sync
```

### Usage

Start worminator:

```bash
python main.py
```
